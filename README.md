## Setup .env file

Set up the environment variables given in the .env.sample file.


## Run Locally

Install dependencies

```bash
  npm install
```

After adding your DATABASE_URL to the .env file, synchronize your database schema by running:

```bash
  npm run db:push
```

Start the app

```bash
  npm run dev
```
